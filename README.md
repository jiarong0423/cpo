# 光通訊產業 AI 供應鏈心智圖 2026

## 📊 項目說明

這是一個互動式可視化應用，展示 2026 年光通訊產業與 AI 供應鏈的完整生態。涵蓋美股垂直整合企業、台灣上中下游廠商，以及 Nvidia 戰略投資的關鍵節點。

## 🚀 快速開始

1. 在瀏覽器中開啟 `index.html`
2. 點擊左側分支以展開/收合公司列表
3. 點擊任何公司卡片查看詳細信息
4. 使用滑鼠滾輪放大/縮小，拖動平移視圖
5. 點擊「下載 SVG」按鈕導出高清圖片

- **台股光模組廠**：上詮、光聖、華星光、光環、眾達

### 中下游

### 下游
- **終端應用**：Nvidia、AWS、Microsoft、Google、智邦

## 🎨 視覺設計

- **藍色 (US)**：美國上市公司
- **綠色 (TW)**：台灣上市公司
- **橙色 (Nvidia)**：Nvidia 入股或綁定的企業
- **黃色 (⚠)**：關鍵物資缺貨警示
